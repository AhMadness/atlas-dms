# Atlas DMS

Atlas DMS is a private, on-prem document management system for teams that need
structured records, controlled access, scanning, audit history, and local data
ownership.

This public repository is a product overview and GitHub Pages site only. The
Atlas application source code, build scripts, vendor license tooling, private
keys, deployment secrets, customer data, and release packages are not included.

## What Atlas Provides

- Template-driven document records with typed index fields.
- Department, folder, template, and document access controls.
- Browser-based document workspace with previews, uploads, versions, merge,
  split, page insertion, and document history.
- Workstation scan agent for browser scanning from each user's Windows PC.
- Storage volume support for adding new disks while keeping old files readable.
- Admin console for users, groups, departments, templates, sessions, audit,
  reports, storage, and licensing.
- Offline signed, machine-locked license model for controlled trials and
  on-prem deployments.
- English and Arabic UI support.
