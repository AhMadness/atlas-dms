# Security

Do not report sensitive customer data, private server details, license keys, or
secrets in public GitHub issues.

For security concerns, contact the Atlas vendor privately with:

- a short summary;
- affected version or deployment type;
- reproduction steps when safe to share;
- impact and suggested mitigation.

Atlas is designed for controlled on-prem deployments with private source,
machine-locked licensing, local storage, audit logging, and administrator-owned
access control.
